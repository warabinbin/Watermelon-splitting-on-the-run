# Watermelon-splitting-on-the-run

# 概要
逃げ回るスイカをたたき割るゲームです。<BR>スイカは飛んだり跳ねたり透明になったりあらゆる手段で逃げます。<BR>


https://user-images.githubusercontent.com/64608456/187064871-318ec6da-6b4e-4719-9038-282d43245176.mp4


  
# システム
  制限時間：30秒 <BR>
  逃げきれたらスイカの勝ち<BR>
  たたき割られたら人類の勝ち<BR>
  
# 装置
 ・HTC VIVE PRO Eye<BR>
  
# スイカ
・移動(WASD)<BR>
・ジャンプ(Enter)<BR>
・走り（スタミナあり）<BR>
・特殊能力：透明化(E)<BR>

# 種類
①緑黒スイカ（デフォルト）<BR>
②赤黒スイカ（スタミナ少ないがスピード速い）<BR>
③青黒スイカ（スタミナ多いがスピード遅い）<BR>
④メロン（最強）<BR>
  
# フィールド
・2m × 2m　<BR>
・加速床　<BR>
・ワープホール　<BR>
